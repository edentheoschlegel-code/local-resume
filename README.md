# LocalResume

A private, in-browser resume builder with real-time resume scoring and job-description matching.

**Live:** [localresumeapp.com](https://localresumeapp.com)

## Highlights
- 100% on-device: career data is never transmitted anywhere
- Offline-first PWA — edit and score resumes without a connection
- Strict Content Security Policy; scoring and job-match analysis run client-side
- One-time Pro (ATS-ready Word export, cover-letter builder) via Stripe + RevenueCat Web Billing
